# Student Management System
A simple Java application to manage and display basic student information.
## Description
This is a beginner-friendly Java program designed to accept and display details for two students. It takes inputs such as Name, Roll Number, and City directly from the user and prints them back on the screen. The code is written in a straightforward, sequential manner without using complex structures like loops, arrays, or advanced collections, making it perfect for beginners to understand and explain.
## Features
* Accepts input for Student Name, Roll Number, and City.
* Displays the collected details cleanly in the console.
* Built using basic Java `Scanner` functionality for easy comprehension.
